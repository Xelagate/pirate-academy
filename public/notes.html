Слайд 1 · Проблема

Solana росте. Шлях зламаний.

  • 17,700 активних девів — це дефіцит кадрів, не ринок. Екосистема задихається
  • Люди йдуть не тому що складно — йдуть бо не бачать прогресу
  • +29% YoY — ріст є, а вчитися нормально ніде
Слайд 2 · Конкуренти

Ніхто не дав Solana свого LeetCode

  • On-chain proof — це не фіча, це новий стандарт резюме у Web3
  • Alchemy, RiseIn, Cookbook — ні в кого немає зв'язки: редактор + гейміфікація + on-chain + Solana
  • Ми перші хто зібрав усе в одному
Слайд 3 · Продукт

Живий. Прямо зараз.

  • Не макап — реальний продукт, можна відкрити під час пітчу
  • Monaco editor — це той самий VS Code у браузері. Поріг входу = нуль
  • Закінчив курс → NFT у гаманець. Твій доказ навички — назавжди
Слайд 4 · User Journey

Від нуля до NFT за одну сесію

  • Не потрібен локальний Rust, Solana CLI, нічого. Браузер + гаманець = старт
  • Конкретна обіцянка: підключив гаманець — через годину NFT в ньому

  • Хакатон — найкращий фільтр. Або робиш, або ні
