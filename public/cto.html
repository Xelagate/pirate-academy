Tech Stack

Стек

Чому саме це, а не щось інше

FRONTEND

Next.js 16 App Router

21 сторінка статично генерується при білді. Zero cold starts, CDN-роздача по всьому світу.

SOLANA SDK

@solana/kit v6

Новий web3.js 2.0. Функціональний API, кращий tree-shaking, повна типізація. Не legacy web3.js.

WALLET

Wallet Standard

Кастомний adapter без залежностей. Phantom, Backpack, Solflare — один інтерфейс для всіх.

EDITOR

Monaco Editor

Той самий рушій що в VS Code. Повна підсвітка синтаксису Rust прямо в браузері.

SMART CONTRACT

Anchor Framework

Дві інструкції: register_pirate і mint_badge. PDA для профілю, CPI в Metaplex.

NFT STANDARD

Metaplex Core

Один акаунт на NFT — без окремого metadata account. Дешевше в SOL-rent, сучасний стандарт.

CODE GEN

Codama

IDL → TypeScript клієнт автоматично. Типобезпечні хелпери для кожної інструкції.

INFRA

Vercel

Static deploy, глобальний CDN, auto-deploy. Операційних витрат = $0 на поточному трафіку.

Code Validation

Як перевіряється код

Клієнтська regex-валідація — свідоме рішення

1

Юзер пише код у Monaco Editor

Повноцінний редактор з підсвіткою, але без компілятора в браузері

2

Натискає "Run" — regex перевіряє паттерн

Кожен урок перевіряє конкретну синтаксичну конструкцію:
/name\s*:\s*String\s*[,}]/ → урок 1

3

Миттєвий фідбек без сервера

Правильно → звуковий сигнал + прогрес-бар + наступний урок відкривається.
Неправильно → конкретна підказка що саме не так.

4

Чому не справжня компіляція?

Fill-the-blank завдання перевіряють розуміння паттерну, не логіку програми. Regex дає інстантний фідбек і не потребує сервера. Для MVP це правильний trade-off.

Key Decisions

Чому так, а не інакше

Архітектурні рішення і їх обґрунтування

localStorage для прогресу

Нема авторизації, нема БД, нема latency. On-chain NFT — це canonical proof навички, він у гаманці назавжди. localStorage — лише UI-зручність.

Навмисне

@solana/kit замість web3.js

Нова функціональна бібліотека (web3.js 2.0). Кращий TypeScript, менший bundle, активно підтримується Solana Foundation. Старий web3.js deprecated.

Правильно

Metaplex Core замість Token-2022

Один акаунт на NFT без окремого metadata акаунту. Простіше, дешевше в SOL-rent, Metaplex Core — сучасний стандарт NFT на Solana.

Правильно

Статична генерація всіх 21 сторінки

generateStaticParams при білді. Жодного SSR, жодного сервера. Vercel CDN роздає статику глобально — максимальна швидкість, мінімальна вартість.

Правильно

Кастомний Wallet adapter

Без @solana/wallet-adapter — важка залежність з legacy web3.js всередині. Власна реалізація Wallet Standard: легше, немає конфліктів версій.

Trade-off

Scalability

Як масштабується

Додати острів = додати файли. Без деплою смарт-контракту.

ДОДАТИ ОСТРІВ

1. MDX-файли для уроків
2. Конфіг острова в lessons.ts
3. Regex-валідатор для кожного уроку
4. Контрактний deploy не потрібен

ПОТОЧНІ ЛІМІТИ

Devnet → Mainnet: зміна одного env var
Прогрес: localStorage → DB при потребі auth
Validation: regex → WASM-компілятор в майбутньому
Усі ліміти — заплановані trade-off MVP

Bottom line: Архітектура навмисно проста. MVP-рішення задокументовані і зрозумілі. Кожен компроміс — свідомий, а не випадковий. Готовий обговорити будь-яке.
