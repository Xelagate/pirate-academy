## Lesson 1 — Structs & Types

Every pirate needs a name. In Rust, you store data in a **struct** — a named collection of fields.

```rust
struct Pirate {
    name: String,
}
```

Fill in the blank: what type should `name` be?

> **In Solana terms:** account state is just a struct. Fields live on-chain in the account's data buffer.
